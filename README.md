# Landfall

学習記録は最小、月末の「学習Wrapped」が主役の学習ログアプリ。

休んだ日数を隠さず、学んだ日数と同格に堂々と見せる。
評価軸は連続日数ではなく**再開力**。やめた回数は、常に0回。

## コンセプト

- ストリーク・連続日数・催促通知は作らない
- 空白(休んだ期間)は物語の一部として肯定的に扱う
- 2日以上の空白のあとに戻ってきた日は「帰還」として讃える

## 画面

| 画面 | 役割 |
| --- | --- |
| 今日 | 学習項目(教材・本・活動)のタイルが並ぶ。タップしてタイマー計測または時間を選んで刻む(+任意のひとこと)。2日以上の空白明けだけ、控えめな「おかえり」演出 |
| 軌跡 | 当月をスカイライン状の波形で一望。空白バーと帰還マーカー、統計3つ(累積/再開/やめた回数=常に0) |
| Wrapped | 月末〜翌月に生成できる4枚のカード(事実/空白の物語/タイプ診断/軌跡の一枚絵)。縦スワイプ、各カード共有可 |

### 学習項目

- タイルは長押しドラッグで並べ替え。「+」から追加
- 見た目は配色×シンボルのプリセット、または教材の表紙写真
- どれかの項目で記録した日が「学んだ日」— 軌跡・Wrapped・再開力は日ベースのまま

「今日」画面右上の設定から、アプリアイコンを **ミッドナイト / コーラル** の2種で切り替えられます。

## タイプ診断(月のデータから機械的に判定、全タイプ肯定的)

| タイプ | 条件 | 決め台詞 |
| --- | --- | --- |
| 不死鳥型 | 5日以上の空白から帰還 | 深く沈み、必ず戻る。 |
| 石橋型 | 空白がすべて2日以内 | 静かに、確実に、積む。 |
| 波乗り型 | 週単位で周期的に学習 | あなたには、あなたの潮がある。 |
| 彗星型 | 学習日の70%以上が連続塊 | 燃えるときは、一気に。 |
| 朝凪型 | 上記以外 | 騒がず、焦らず、途切れず。 |

## 技術

- SwiftUI + SwiftData / iOS 17+
- 完全ローカル(バックエンド・アカウント・通知・計測SDKなし)
- 共有画像は `ImageRenderer` でカードをPNG化 → `ShareLink`

## ビルド

```
open Landfall.xcodeproj
```

Xcode 16 以降でそのままビルドできます(依存パッケージなし)。

## 構成

```
Landfall/
  LandfallApp.swift        # エントリ(タブ: 今日/軌跡/Wrapped)
  Design/Theme.swift       # パレット・タイポ・カード共通部品
  Models/                  # StudyDay(SwiftData) / WrappedMonth(派生値) / 診断
  Views/
    Today/  Trace/         # 今日・軌跡
    Waveform/              # スカイライン波形(再利用部品)
    Wrapped/               # カード4枚・シンボル・共有
Tools/RenderHarness/       # カードをPNG化する確認用CLI(アプリ外)
```
